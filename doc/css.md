# Cascading Style Sheet

## BEM (Block Element Modifier)

`id`와 `class`를 섞어쓰면 헷갈리니까 `class`만 쓴다.

Element가 포함관계를 가질 때는 `__`로 이어준다.

Modifier를 가질 경우, `--`로 이어준다.

```css
/* Example */

.btn {}
.btn__price {}
.btn--orange {}
```
